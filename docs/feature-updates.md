# Server Release Notes

`server` 모듈의 정식 릴리스 기능 명세입니다.

---

## [v1.2.0] - 2026-03-23 🚀

서비스 브랜딩을 '전북대 수강신청 도우미'에서 **'줍줍 (zup-zup)'**으로 전면 개편했습니다.

### Added
- **도메인 설정**: `www.zup-zup.com` 도메인 연동을 위한 CORS 정책 및 OAuth 리다이렉트 URI 반영 (`application-prod.yml`)

### Changed
- **앱 명칭 변경**: `application.yml` 내 `spring.application.name`을 `zup-zup`으로 변경
- **알림 타이틀**: 발송되는 알림의 말머리를 `[전북대 수강신청 도우미]` ➡️ **`[줍줍]`** 으로 변경
- **크롤러 식별자**: K6 성능 테스트용 User-Agent 명칭 변경 (`Zup-Zup-Bot`)

---

## [v1.1.0] - 2026-03-09 🚀

강의 리뷰 시스템 고도화 및 통합 건의/제보 백엔드 인프라가 구축되었습니다.

### Added

- **통합 건의/제보(Feedback) 시스템**: 사용자 피드백 수집 및 관리자 답변 처리를 위한 도메인 모델(Entity, Repository) 구축
- **안전한 멀티파트 파일 업로드**: `Apache Tika` 라이브러리를 활용한 파일 시그니처 정밀 검증 및 UUID 기반 로컬 저장 시스템 구현
- **관리자 액션 로그(AdminActionLog)**: 상태 변경, 답변 등록 등 관리자의 주요 행위를 추적하기 위한 데이터 감사 시스템 도입
- **데이터베이스 스키마 확장**: 피드백, 첨부파일, 답변 관리용 테이블 추가 반영 (V2 migration 최적화)
- **강의 리뷰(Review) 시스템**: 강의별 1~5점 별점 및 익명 리뷰 작성 기능 구현
- **리뷰 반응(Reaction) 기능**: 리뷰별 공감(LIKE) / 비공감(DISLIKE) 토글 시스템 구축
- **사용자 맞춤형 리뷰 정렬**: 리뷰 목록 조회 시 현재 로그인한 사용자의 리뷰를 최상단에 자동 고정하는 쿼리 로직 반영
- **리뷰 통계 자동 집계**: 리뷰 작성/삭제 시 해당 강의의 평균 별점 및 리뷰 수 실시간 자동 갱신

### Changed

- **로컬 개발 데이터 영속화**: H2 데이터베이스를 `file` 모드로 전환하여 서버 재시작 시에도 데이터가 유지되도록 구성 (`./data/jbnu-sugang`)
- **인증 로직 고도화**: `SecurityContextHolder`를 활용한 서비스 레이어 내 인증 정보 획득 방식 정규화
- **DTO 명세 일관성**: 모든 Response DTO에서 Fully Qualified Name 제거 및 표준 Import 방식 통일
- **테스트 커버리지 및 패턴 최적화**: `Given/When/Then` 기반의 BDD 테스트 패턴 적용 및 중복 모킹 로직 helper 메서드화
- **로그 및 에러 메시지 표준화**: 모든 로그 메시지를 영문으로 통일하고 예외 메시지의 가독성 개선
- **소프트 삭제(Soft Delete) 정책**: 피드백 및 답변 데이터 삭제 시 물리적 삭제 대신 `deleted_at` 컬럼을 활용한 논리 처리 적용

## [v1.0.0] - 2026-03-01 ✅

줍줍 서버의 모든 핵심 기능을 통합한 정식 릴리스입니다.

### Added

- **OASIS 크롤링 파이프라인**: 전북대 오아시스 XML 수집 및 파싱 클라이언트 구축
- **멀티 채널 알림 엔진**: EMAIL, Web Push(FCM), Discord 연동 발송기 구현
- **실시간 여석 감지**: `SeatOpenedEvent` 기반의 비동기 알림 트리거 시스템
- **알림 중복 방지(Dedup)**: Redis 기반 10분 이내 동일 과목 알림 차단 로직 적용
- **공지사항(Announcement) 시스템**: 제목/내용 통합 검색 및 목록/상세 DTO 분리 조회 API
- **크롤링 타겟 동적 관리**: 관리자 전용 연도/학기 설정 및 수동 크롤링 트리거 엔드포인트
- **정밀 시간 검색 모델**: `dayOfWeek`, `startTime`, `endTime` 기반의 시간 범위 검색 엔진
- **인증 및 온보딩**: Google OAuth2 연동 및 및 Refresh Token 보안 세션 관리
- **관리자 대시보드 API**: 시스템 가입자, 알림 발송 현황 통계 및 실시간 로그 조회

### Changed

- **DB 마이그레이션 통합**: 산재된 스키마 변경 이력을 `V3__consolidate_schema_updates.sql`로 최적화 및 통합
- **테스트 알림 정책**: 사용자별 10초 쿨타임 제한 및 운영 통계(히스토리) 제외 처리 적용
- **도메인 모델 정규화**: 불필요한 `period` 필드 제거 및 시간표 검색 로직 고도화
